## Featured Project

### Hayai — Database Management CLI

A fast and extensible CLI tool for managing local development databases through Docker. Designed for backend developers who need reliable database instances configured in seconds rather than hours.

- **Stack:** TypeScript, Node.js, Docker, Docker Compose
- **Coverage:** 19+ databases across relational, NoSQL, vector, and time-series categories — including PostgreSQL, Redis, InfluxDB, Qdrant, Meilisearch, and TimescaleDB
- **Features:** Interactive CLI, automatic configuration, reproducible environments

Repository: [github.com/hitoshyamamoto/hayai](https://github.com/hitoshyamamoto/hayai) · NPM: [hayai-db](https://www.npmjs.com/package/hayai-db)
