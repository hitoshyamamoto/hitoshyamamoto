## About Me

I work on **systems integration, network architecture, and solutions architecture** at **Rennosonic Tecnologia**, building IIoT infrastructure for industrial monitoring across critical-infrastructure sites: edge data acquisition over Modbus, a Sparkplug B unified namespace on MQTT, time-series historians, and Grafana-based monitoring across dozens of remote sites. This includes designing and certifying the **OT/IT network layer** itself: segmenting trust zones along the **Purdue model**, applying **IEC 62443** principles, and managing firewall and VLAN architecture so that field, control, and enterprise systems integrate securely.

I'm concluding my B.S. in **Computer Engineering at the Federal University of Itajubá (UNIFEI)**, with a prior technical degree as a **Mechatronics Technician at ETEP**. Before Rennosonic, as a Development Analyst at **Wonder Datalabs**, I led integrations with major LLM providers and designed **agentic AI workflows** for production platforms. I continue to build on that work across orchestration, tool use, and agent design.

What ties it together is a focus on the unglamorous, high-leverage layers of software: secure, reliable integration between heterogeneous systems, clean APIs over messy industrial reality, and tooling that other engineers can actually depend on. I publish that tooling as open source.

## Featured Projects

### soapbar — Modern SOAP library for Python

A clean, typed SOAP stack for Python. Build servers with a `SoapService` class and `@soap_operation` decorators (with auto-generated WSDL), or drive a typed client from an existing WSDL. Supports SOAP 1.1/1.2 and all binding styles. Built to handle the demands of real-world industrial standards.

- **Stack:** Python · Apache-2.0
- Repository: [github.com/hitoshyamamoto/soapbar](https://github.com/hitoshyamamoto/soapbar) · PyPI: [soapbar](https://pypi.org/project/soapbar/)

### hayai — Database management CLI

A fast, extensible CLI for spinning up local development databases through Docker. For backend developers who want reliable instances configured in seconds, not hours.

- **Stack:** TypeScript · Node.js · Docker · Docker Compose
- **Coverage:** 19+ databases across relational, NoSQL, vector, and time-series categories (PostgreSQL, Redis, InfluxDB, Qdrant, Meilisearch, TimescaleDB, and more)
- Repository: [github.com/hitoshyamamoto/hayai](https://github.com/hitoshyamamoto/hayai) · NPM: [hayai-db](https://www.npmjs.com/package/hayai-db)
